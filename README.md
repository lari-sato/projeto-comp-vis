# 🏞️ Projeto 1 - Processamento de imagens

## 🎯 Objetivo
Desenvolver um software de processamento de imagens em linguagem C utilizando a biblioteca SDL (Simple DirectMedia Layer)

## 🔧 Funcionamento

## 👩🏻‍💻 Compilação e execução
- Abra um terminal dentro da pasta do projeto
- Para compilar, utilize ```gcc projeto.c -o projeto```
- Para executar, utilize ```./projeto <caminho da imagem.extensao>```
